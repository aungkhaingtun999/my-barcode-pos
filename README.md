# my-barcode-pos
An efficient Barcode POS system featuring real-time inventory tracking and seamless sales management, powered by Streamlit and integrated with Supabase for secure data storage.
Barcode POS System
A web-based Barcode POS (Point of Sale) system built with Python and Streamlit. Designed for efficient inventory management and fast checkout processes.

Features
Real-time Inventory Tracking: Keep your stock updated automatically.

Barcode Scanning: Fast and accurate product scanning for checkout.

Secure Database: Powered by Supabase for reliable and secure data storage.

Streamlit Interface: User-friendly and responsive design.

Prerequisites
To run this project, you need to have:

Python 3.9 or higher installed.

A Supabase account and project API keys.

Deployment
This app is deployed using Streamlit Community Cloud.
